Annotators

Annotators are codes that run on the Lasair transient stream, compute some value added feature(s) for the transients and inject these values back into the stream for other users to use in their filters. Some annotators are written by the Lasair Team but others are submitted by a Lasair users. Please get in contact via Community if you have an idea for an annotator.
